# Videos Folder

## Advertisement Video

Place your advertisement video here with the name: `ad-placeholder.mp4`

**Specifications:**
- Format: MP4
- Resolution: 950 x 250 pixels
- Codec: H.264
- Autoplay: Yes (muted)
- Loop: Yes

The video will be displayed in the advertisement card on category pages (e.g., /categories/ai-porn-chat-platforms) and will appear after the 3rd companion in the list.

## Current Implementation

The advertisement card includes:
- Video player (950x250px)
- "Learn More" CTA button (blue)
- "Visit Website" CTA button (outlined)
- "Sponsored" badge in top-right corner

The card has the same styling and spacing as regular companion cards to blend naturally into the grid.
