Meta Meta Pixel Events

Events tracked via Meta Pixel (fbq) for Facebook/Instagram advertising optimization and audience building.

# Event Name Type Description Trigger Source File
1 PageView Standard Automatic page view tracking. Fires on every page load. Used for building retargeting audiences and measuring traffic from Meta ads. Automatically on every page load (inline script in <head>) Inline in HTML <head>
2 ViewContent Standard Tracks when users view specific content pages (companion reviews, news articles). Includes content name, category, and IDs for product catalog integration. Page load on /companions/* or /news/* pages js/meta-companion-tracking.js
3 Search Standard Tracks search queries entered by users. Captures the search string for understanding user intent and optimizing search functionality. User presses Enter in a search input field js/meta-companion-tracking.js
4 Lead Standard Tracks lead generation events: contact form submissions, review submissions, and floating CTA clicks. Key conversion event for Meta ad optimization. Contact form submit, review form submit, or COTM/Featured floating CTA clicks js/meta-form-tracking.js
js/floating-cta.js
5 CompleteRegistration Standard Tracks successful newsletter signups and registrations. Fires when a user completes the newsletter subscription process. Newsletter form submission (via form or checkbox in contact form) js/meta-form-tracking.js
6 CompanionExternalClick Custom Custom event tracking all external link clicks with companion context. Similar to GA4 outbound_click but optimized for Meta's event structure. Click on any external link with target="_blank" js/meta-companion-tracking.js
7 AdClick Custom Custom event for tracking advertisement clicks specifically. Provides ad placement and click type context for measuring ad performance. Click on ad placement element js/ad-tracking.js

Tracking Files Overview

JavaScript files responsible for event tracking

File Platform Purpose Events
js/ga-external-tracking.js GA4 Main GA4 tracking script. Handles outbound link tracking with Airtable enrichment, scroll depth, form submissions, CTA clicks, and enhanced page views. outbound_click scroll form_submit cta_click page_view
js/ad-tracking.js GA4 Meta Dedicated ad click tracking. Uses event delegation to capture clicks on ad placements and sends to both GA4 and Meta Pixel. ad_click (GA4) AdClick (Meta)
js/meta-companion-tracking.js Meta Meta Pixel tracking for companion-related events. Handles ViewContent for companion/article pages, search tracking, and external link clicks. ViewContent Search CompanionExternalClick
js/meta-form-tracking.js Meta Meta Pixel form tracking. Captures contact form submissions as leads and newsletter signups as registrations. Lead CompleteRegistration
js/newsletter-signup.js GA4 Newsletter subscription handler. Submits to Netlify function and tracks successful signups in GA4. newsletter_signup
js/companion-floating-cta.js GA4 Meta Floating CTA on companion pages. Shows current companion + "Best Alternative" button linking to high-rated companions (Hammer AI, Secrets AI, Soulkyn AI). outbound_click cotm_floating_cta_click best_alternative_click Lead
js/floating-cta.js GA4 Meta Floating CTA on news articles. Shows COTM or article-specific featured companion with "Full Review" and "Visit Website" buttons. cotm_floating_cta_click featured_floating_cta_click Lead
js/deals.js GA4 Deals page dynamic loading. Fetches active deals from Airtable and tracks deal CTA clicks. deal_click
faq-interactions.js GA4 FAQ accordion functionality with accessibility support. Tracks which FAQ questions users open/close. faq_interaction
